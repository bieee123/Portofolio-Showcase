Academic Project

This project was conducted as part of the Cyber Security: Network Security Fundamentals bootcamp by TalentHub. Completing this project was a requirement for graduation, showcasing skills in vulnerability assessment, identification of security risks, and providing mitigation strategies.

The project involved assessing the security posture of a target website by utilizing industry-standard tools and methodologies. It emphasized practical application of concepts learned during the bootcamp, such as information gathering, vulnerability scanning, manual analysis, and report writing.

1. Introduction

Assessment Details

Assessment Date: July 25, 2024
Assessor: Gabrielle B. Lintong
Target: http://34.45.20.126

Objective

To identify and evaluate security vulnerabilities in the target website through comprehensive security testing and analysis.

2. Methodology

Information Gathering

Using tools like Nmap to collect basic data about the target system and identify open ports and services.

Vulnerability Scanning

Utilizing OpenVAS to identify existing vulnerabilities through automated scanning techniques.

Manual Analysis

Verifying findings using Nikto for in-depth manual analysis and validation of discovered vulnerabilities.

Reporting

Compiling a comprehensive report on findings, impacts, and mitigation recommendations.

3. Findings

Critical Vulnerabilities

Remote Code Execution (RCE)

Description

This vulnerability allows attackers to execute arbitrary code on the target system remotely, potentially giving them complete control over the affected server.

Potential Impact
  • Complete system takeover and unauthorized access
  • Sensitive data theft and information disclosure
  • Malware distribution and backdoor installation
  • Service disruption and denial of service attacks

General Recommendations

Security Improvement Strategies

Software Updates

Update all software components to the latest stable versions with security patches.

Server Configuration

Configure the server with robust security settings and hardening guidelines.

Regular Audits

Conduct regular security audits and penetration testing to identify new vulnerabilities.

Security Tools

Utilize vulnerability detection and mitigation tools for continuous monitoring.

4. Conclusion

"This assessment identified several critical vulnerabilities requiring immediate attention. Implementing the recommended mitigations is expected to significantly enhance the security of the target website and protect against potential cyber threats."
— Assessment Summary

Key Takeaways

  • Immediate patching of critical vulnerabilities is essential
  • Regular security assessments help maintain robust defenses
  • Proper configuration management reduces attack surface
  • Continuous monitoring enables early threat detection
