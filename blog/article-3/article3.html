Vulnerability Assessment

Gabrielle Briliant Lintong Cyber Security Researcher

This project was conducted as part of the Cyber Security: Network Security Fundamentals bootcamp by TalentHub. Completing this project was a requirement for graduation, showcasing skills in vulnerability assessment, identification of security risks, and providing mitigation strategies.

The project involved assessing the security posture of a target website by utilizing industry-standard tools and methodologies. It emphasized practical application of concepts learned during the bootcamp, such as information gathering, vulnerability scanning, manual analysis, and report writing.

1. Introduction

  • Assessment Date: July 25, 2024
  • Assessor: Gabrielle Briliant Lintong
  • Target: http://34.45.20.126
  • Objective: To identify and evaluate security vulnerabilities in the target website.

2. Methodology

  • Information Gathering: Using tools like Nmap to collect basic data about the target.
  • Vulnerability Scanning: Utilizing OpenVAS to identify existing vulnerabilities.
  • Manual Analysis: Verifying findings using Nikto for in-depth manual analysis.
  • Reporting: Compiling a report on findings, impacts, and mitigation recommendations.

3. Findings

Vulnerabilities

  • Vulnerability: Remote Code Execution (RCE)
    • Description: This vulnerability allows attackers to execute arbitrary code on the target system remotely.
    • Impact: Could lead to complete system takeover, data theft, and malware distribution by attackers.

General Recommendations

  • Update software to the latest version.
  • Configure the server with robust security settings.
  • Conduct regular security audits.
  • Utilize vulnerability detection and mitigation tools.

4. Conclusion

This assessment identified several critical vulnerabilities requiring immediate attention. Implementing the recommended mitigations is expected to significantly enhance the security of the target website.
